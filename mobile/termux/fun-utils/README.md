# 🎉 Fun & Toys

> Because terminals deserve personality: matrix rain, fake hacker mode, fortune cookies.

---

## Tools in this category

- [`matrix`](./matrix/) — Matrix rain effect in your terminal
- [`hack`](./hack/) — Fake 'hacking' terminal animation for demos
- [`fortune-cookie`](./fortune-cookie/) — Print a random fortune-cookie message
- [`banner`](./banner/) — Print a large ASCII banner from text
- [`coin-flip`](./coin-flip/) — Flip a coin (heads or tails)
- [`dice`](./dice/) — Roll N dice with M sides each (default 1d6)
- [`cowsay`](./cowsay/) — Make a cow say something (no Perl required)
- [`figlet-small`](./figlet-small/) — Small figlet-style banner (single-line)
- [`yes`](./yes/) — Repeatedly print a string until killed (like yes(1))
- [`typing-test`](./typing-test/) — Tiny typing speed test (10 short words)

## Installation

Each tool is self-contained. Either copy the script into your `$PATH` (e.g. `~/.local/bin/`) or use the global installer at the repository root:

```bash
./install.sh
```

## Usage pattern

```bash
# from the repo root
./fun-utils/<tool-name>/<tool-name>.sh   # or .py

# or, after install.sh
<tool-name>
```
