# 🖥️ Terminal Utilities

> Profile, theme, and tune the terminal itself: colors, fonts, prompt segments.

---

## Tools in this category

- [`term-size`](./term-size/) — Print current terminal rows and columns
- [`term-reset`](./term-reset/) — Reset the terminal (clear + reset colors + cursor)
- [`term-title`](./term-title/) — Set the terminal window/tab title
- [`cursor-hide`](./cursor-hide/) — Hide the terminal cursor (undo with cursor-show)
- [`cursor-show`](./cursor-show/) — Show the terminal cursor
- [`theme-preview`](./theme-preview/) — Print a preview of common terminal color codes

## Installation

Each tool is self-contained. Either copy the script into your `$PATH` (e.g. `~/.local/bin/`) or use the global installer at the repository root:

```bash
./install.sh
```

## Usage pattern

```bash
# from the repo root
./terminal-utils/<tool-name>/<tool-name>.sh   # or .py

# or, after install.sh
<tool-name>
```
